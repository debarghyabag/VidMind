# VidMind Edge Extension

This extension calls the deployed VidMind API from the current YouTube tab.

## Load in Microsoft Edge

1. Open `edge://extensions`.
2. Turn on **Developer mode**.
3. Choose **Load unpacked**.
4. Select this `extension` folder.
5. Open a YouTube video and click the VidMind extension icon.

The popup defaults to:

```text
https://vidmind-45l8.onrender.com
```

Use the settings button in the popup to switch to a local backend:

```text
http://127.0.0.1:8000
```
